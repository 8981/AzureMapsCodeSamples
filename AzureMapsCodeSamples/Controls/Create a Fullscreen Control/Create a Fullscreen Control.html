Create a Fullscreen Control This sample shows how to create a custom fullscreen control that can be added to the map. In this case a control for toggling the map between its specified size in the page and fullscreen mode. This control aligns its style with the built in controls in Azure Maps and also provides additional styling functionality to allow the controls color to be a CSS3 color or have it automatically switch between light and dark based on the map style. A framework for supporting translated text for aria-labels (accessibility) has also been added.
